# Pamela Bakery - Pastelería Artesanal Premium

Proyecto web para una pastelería artesanal, con un diseño femenino, moderno, cálido y premium.
Desarrollado con Next.js 14, Tailwind CSS y Framer Motion.

## 🚀 Instalación y Desarrollo Local

Como este proyecto fue generado manualmente, asegúrate de seguir estos pasos para correrlo en tu máquina (MacOS):

### 1. Requisitos previos
Debes tener instalado **Node.js** (versión 18 o superior). Puedes descargarlo desde [nodejs.org](https://nodejs.org/) o instalarlo vía Homebrew si usas Mac:
```bash
brew install node
```

### 2. Instalar dependencias
Abre tu terminal en la carpeta del proyecto (`/Users/gibarra/Desktop/PamelaWeb`) y ejecuta:
```bash
npm install
```

### 3. Correr el servidor de desarrollo
Para ver la web localmente, ejecuta:
```bash
npm run dev
```
Luego, abre tu navegador en [http://localhost:3000](http://localhost:3000).

## 🌐 Deploy en Vercel

Esta web está optimizada y lista para producción en Vercel.

1. Entra a [Vercel](https://vercel.com/) e inicia sesión con tu cuenta de GitHub/GitLab.
2. Haz clic en **"Add New..."** > **"Project"**.
3. Importa el repositorio de **PamelaWeb**.
4. Vercel detectará automáticamente que es un proyecto **Next.js**.
5. Haz clic en **Deploy**. ¡Y listo! Tu web estará online en un par de minutos.

## ✨ Características Técnicas

- **Next.js App Router**: Estructura de componentes del lado del servidor y del cliente optimizada.
- **Tailwind CSS**: Estilado premium con paleta de colores personalizada (pastel).
- **Framer Motion**: Animaciones delicadas y fluidas (scroll reveals, hero fade-in, microinteracciones).
- **SEO Ready**: Etiquetas OpenGraph y metadatos configurados para destacar en búsquedas.
- **WhatsApp Dinámico**: Cada producto genera un mensaje personalizado listo para enviar.
- **Diseño Responsive**: Perfecto en móviles, tablets y monitores grandes.
